# 露水瓶

<div align=center><img src=../../resources/icon/dew_bottle-128px.png></div>

​     

| 添加此物品的原因 | 李芒果空岛前期需要容器来收集露水 |
| :--------------- | :------------------------------- |
| 稀有度           | 常见                             |
| 命名空间         | comfysky:dew_bottle              |
| 添加版本         | 17.0.3                           |

​     

## 获取

玻璃瓶(空)右键露水花转换为露水瓶

​     

## 用途

采集露水

​     

## 交互

1.手持玻璃瓶(空)右键露水花采集露水

2.手持耐久度为50的露水瓶右键炼药锅，返还空瓶，并增加炼药锅1/3的水

​
